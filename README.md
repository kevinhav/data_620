# DATA 620: Web Analytics

CUNY School of Professional Studies - M.S. in Data Science

## Course Overview

Analysis of website interactions, mobile data, and social networks. Covers social network analysis (centrality, cohesion, affiliations, clustering coefficient), network structures, website traffic analysis (click streams, referrals, keywords, page views, drop rates), search algorithms, and SEO fundamentals.
